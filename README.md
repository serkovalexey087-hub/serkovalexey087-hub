🎓 Дипломный проект: Архитектура AI-агентов для обработки мультимедиа
Технологический стек: Python Streamlit BPMN LLM MoviePy LangChain
Создаю систему автоматизированной обработки видео и текстовых данных через пайплайн независимых AI-агентов:
✅ Модульная архитектура — каждый агент принимает решения и передаёт результат следующему этапу
✅ Визуализация процессов — диаграммы BPMN для прозрачности бизнес-логики
✅ Локальная обработка — работа с русскоязычными моделями, минимизация зависимости от облака
✅ Fallback-механизмы — ручные процессы как резерв при сбоях
